# OpenBB 数据源概览

OpenBB 接入的 A 股、港股、美股数据源可按免费（含基础免费 + 限制免费）、付费（含增值付费 + 机构付费）两个梯度分类归纳，其中免费数据源多靠插件扩展适配，付费数据源则以机构级深度数据为主。

## A 股

| 类型 | 数据源 | 详情 |
| :--- | :--- | :--- |
| 免费 | AKShare | 通过 `openbb_akshare` 插件接入，完全免费，整合东方财富、同花顺等多个平台公开数据，覆盖 A 股行情、财务、舆情等基础数据，适配国内用户无需 VPN。 |
| 免费 | Baostock | 开源免费，提供 A 股完整的历史行情和上市公司财务数据，适合量化交易的基础数据需求，仅更新频率略低于 AKShare。 |
| 免费 | 麦蕊智数 | 支持免费调用，可获取 A 股北交所、科创板实时数据，同时涵盖基本面、资金流及 KDJ、MACD 等技术指标数据，服务稳定性较强。 |
| 限制免费 | Tushare | 通过 `openbb-tushare` 插件接入，免费版功能有限，仅能获取基础行情数据，高级功能如实时高频数据需通过积分兑换解锁，适合个人开发者入门使用。 |
| 付费 | 东方财富 Choice | 机构级付费数据源，集成后可获取 A 股深度行情、机构持仓、行业研报等专业数据，数据更新及时，主要面向企业和机构用户。 |
| 付费 | 同花顺 iFind | 付费接入，提供 A 股精细化财务指标、资金流向、舆情分析等数据，适配复杂量化分析和机构投研场景。 |
| 付费 | Wind | 国内权威机构级数据源，付费集成后能获取 A 股全面的财务报表、分红记录、互联互通资金数据等深度信息，数据准确性和完整性高。 |

## 港股

| 类型 | 数据源 | 详情 |
| :--- | :--- | :--- |
| 免费 | AKShare | 借助插件接入，免费抓取港股基础行情和公司动态数据，数据源自腾讯财经、新浪财经等平台，满足普通投资者基础查询需求。 |
| 免费 | Yahoo Finance | OpenBB 默认基础数据源，免费提供港股基础行情，但覆盖深度不足，且国内用户访问需依赖 VPN，仅适合临时基础查询。 |
| 免费 | 聚合数据 | 每日有 50 次免费调用额度，可获取港股实时交易数据，不过数据维度单一，仅能满足简单行情查看需求。 |
| 付费 | Wind | 付费集成后，弥补免费数据源短板，提供港股完整的财务数据、互联互通资金流向、机构持仓等深度内容，适配机构投研场景。 |
| 付费 | 东方财富 Choice | 机构级付费数据源，可获取港股细分行业数据、公司深度财报及跨境资金动态，支撑港股与 A 股联动分析。 |
| 付费 | 同花顺 iFind | 付费接入后，提供港股精细化的资金流、舆情数据及与 A 股的联动指标，适合跨境投资的量化分析。 |

## 美股

| 类型 | 数据源 | 详情 |
| :--- | :--- | :--- |
| 免费 | Yahoo Finance | OpenBB 默认核心免费数据源，无需额外插件，可获取美股实时行情、历史 K 线、基础财务报表等，满足个人投资者和入门量化需求。 |
| 免费 | Alpha Vantage | 免费版可获取美股股价、技术指标等数据，API 调用次数有一定限制，适配简单策略分析，通过插件即可接入 OpenBB。 |
| 免费 | Google Finance API | 免费提供美股实时报价、技术指标和财经新闻，支持批量获取数据，可通过 OpenBB 统一调用，适合多维度基础数据整合。 |
| 限制免费 | Finnhub | 免费版可获取美股高频基础数据，支持 WebSocket 传输，但 API 调用次数和数据范围受限，高级高频交易数据需付费订阅。 |
| 限制免费 | Tiingo | 基础版免费，能获取美股行情和部分财经新闻数据，若需更高频率的数据更新和更全面的财务维度数据，需升级付费。 |
| 付费 | Polygon | 基础行情可免费获取，高级功能如美股实时明细交易数据、深度盘口数据需付费解锁，适配 OpenBB 量化回测等专业场景。 |
| 付费 | FMP (Financial Modeling Prep) | 基础财务数据免费，美股标准化财务报表、估值指标等深度数据需付费，集成后适配量化策略中基本面因子计算场景。 |
| 付费 | Bloomberg | 通过定制化集成接入，付费获取美股顶级权威数据，涵盖全球跨境资金流向、机构重仓变动、行业深度研报等，是机构美股投研的核心数据源。 |
